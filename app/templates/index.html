Welcome to Immunolyser 1.0


Immunolyser is an automated web-based tool developed for immunologists to analyze immunopeptidomics data in a seamless pipeline. It integrates multiple computational tools for peptide analysis, generating a comprehensive report summarizing sequence conservation, clustering of similar peptides, and predicting binding affinity to various MHC allotypes based on uploaded peptide data. The user-friendly interface requires no programming experience.

For detailed guidelines on using the pipeline and analyzing the web-based output reports, please refer to the help page.

How to use Immunolyser?

Use the Initialiser module to submit a job. The following inputs are required:

  • Sample file(s): A CSV file with a 'Peptide' column. Additional columns like 'Length' and 'Accession' are optional. Multiple samples and replicate files can be uploaded.
  • Control file(s) (optional): A CSV file with a 'Peptide' column. Peptides in this file are labeled as 'control' in results. Multiple control files can be uploaded.
  • Alleles of interest (optional): MHC-I/II alleles for peptide-MHC binding predictions.
  • Running time: Please wait patiently as Immunolyser processes the data. Do not refresh or close the page.
